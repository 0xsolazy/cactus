# cactus

multiplier pool on ethereum. send ether, queue up, get 1.5x back when enough new players join behind you.

## deploy

    npm install
    PASSWORD=hunter2 node deploy.js

## play

send `>=0.5 ether` to the contract. you join the queue. older queued players get paid from the pot as new ether comes in.

owner takes 5% off each entry.
